## Column Width Calculation

For a typical 1280px wide window:
- Available width: 1280px - 120px (margins) = 1160px
- Column width: (1160px - 32px gutters) / 3 ? 376px each

## Responsive Behavior

The layout maintains 3 columns across different window sizes by using `*` (star) sizing for columns, which distributes available space equally.

Minimum recommended width: 1024px
Optimal width: 1280px or larger

At smaller widths, consider switching to a 2-column or single-column layout (not implemented in current version).
