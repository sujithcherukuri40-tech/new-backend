## ?? Quick Start Guide

### For Developers
1. Open `AdminDashboardViewModel.cs`
2. Locate `CreateUserInBackendAsync()` stub
3. Replace with actual API call:
   ```csharp
   return await _adminService.CreateUserAsync(fullName, email, role);
   ```
4. Same for `DeleteUserInBackendAsync()`

### For Testers
1. Navigate to Admin Dashboard
2. Click "? Add User"
3. Test validation (empty fields, invalid email)
4. Create a test user
5. Try deleting users (note disabled states)
6. Verify confirmation dialog

## ?? Code Snippets

### Add User Command
```csharp
[RelayCommand]
private async Task CreateUserAsync()
{
    // Validate
    if (string.IsNullOrWhiteSpace(NewUserFullName))
        NewUserFullNameError = "Full name is required";
    
    // Create
    var success = await CreateUserInBackendAsync(
        NewUserFullName, 
        NewUserEmail, 
        NewUserRole);
    
    // Update UI
    Users.Add(new UserListItem { ... });
    IsAddUserDialogOpen = false;
}
```

### Delete User Command
```csharp
[RelayCommand]
private async Task ConfirmDeleteAsync()
{
    var success = await DeleteUserInBackendAsync(
        UserToDelete.Id);
    
    Users.Remove(UserToDelete);
    IsDeleteConfirmationDialogOpen = false;
}
```
